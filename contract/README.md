## Demo Lazy Minting Contract

### Setup

```shell
# Install dependencies (Truffle, SolC, etc)
$ yarn
```

### Deploy

```shell
# Start Ganache first

$ yarn deploy
```

### Testing

```shell
# Single run
$ yarn test

# Single run with debug
$ yarn test:debug

# Watch for changes
$ yarn test:watch

# Watch for changes with debug
$ yarn test:watch:debug
```
